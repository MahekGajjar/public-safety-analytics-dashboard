# Project Notes

## Data Sources
- Crime and arrest data were sourced from public datasets.
- Jail operations and facility support data were generated to simulate a public safety operations reporting environment.

## Tools Used
- SQL Server Express
- SSMS
- Power BI Desktop
- Google Colab
- Python
- Pandas

## Purpose
This project was built to demonstrate operational data analytics and reporting skills in public safety, crime reporting, arrest analysis, jail operations monitoring, and risk monitoring.
